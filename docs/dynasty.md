# 秦朝至清朝起止年表

| 朝代 / 时期 | 起止年份 | 著名皇帝（举例）                                |
| :---------- | :------- | :---------------------------------------------- |
| 秦朝        | 前221年—前207年 | 秦始皇（嬴政）                                  |
| 西汉        | 前202年—公元9年 | 汉高祖（刘邦）、汉武帝（刘彻）                  |
| 东汉        | 25年—220年      | 汉光武帝（刘秀）                                |
| 三国        | 220年—280年     | 魏武帝（曹操）、蜀汉昭烈帝（刘备）、吴大帝（孙权） |
| 西晋        | 265年—316年     | 晋武帝（司马炎）                                |
| 东晋        | 317年—420年     | 晋元帝（司马睿）                                |
| 南北朝      | 420年—589年     | （多个政权，著名者如北魏孝文帝等）              |
| 隋朝        | 581年—618年     | 隋文帝（杨坚）、隋炀帝（杨广）                  |
| 唐朝        | 618年—907年     | 唐太宗（李世民）、唐玄宗（李隆基）              |
| 五代十国    | 907年—960年     | （略）                                          |
| 北宋        | 960年—1127年    | 宋太祖（赵匡胤）                                |
| 南宋        | 1127年—1279年   | 宋高宗（赵构）等                                |
| 元朝        | 1271年—1368年   | 元世祖（忽必烈）                                |
| 明朝        | 1368年—1644年   | 明太祖（朱元璋）、明成祖（朱棣）                |
| 清朝        | 1636年¹—1912年  | 清圣祖（康熙）、清高宗（乾隆）                  |

> ¹ 清朝建立于1636年（皇太极改国号为清），1644年入主中原。为与常见历史分期一致，也可记为1644年—1912年。
